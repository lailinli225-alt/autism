# 自闭症家庭支持五流派 Skill 库

本文件说明 `src/data/autismInterventionSkills.js` 中的 5 个 Skill。它们用于后续网站集成：用户输入孩子当前表现后，可以分别调用每个 Skill 生成独立分析，再由综合模块汇总。

## 安全边界

- 这些 Skill 是“基于公开理论和资料的模拟分析视角”，不是专家本人在线诊疗。
- 不能输出诊断结论、保证疗效、药物建议或替代个训方案。
- 如出现自伤、攻击、走失风险、明显退行、疑似癫痫、睡眠饮食急剧恶化或家长无法保证安全，应提示尽快联系儿童发育行为、儿童精神心理或急诊专业人员。

## Skill 1：邹小兵 BSR 结构化社交行为视角

**核心用途**：把孩子的问题放回社交发展、结构化支持和行为功能里理解，强调家庭和自然情境中的社交训练。

**理论抓手**

- ASD 干预以社会交往缺陷作为核心任务。
- 干预应科学循证、个性化、家庭和社区为基地。
- BSR 以行为疗法为基本手段，以结构化教育为框架，以人际发展社会交往为基本内容。

**输出重点**

- 可能原因：社交阶梯能力、环境要求、结构化不足、行为功能。
- 处理原则：先降低压力，再创造成功经验，逐步提升要求。
- 干预动作：固定流程、社交游戏、小步成功、记录触发点和恢复时间。

**主要来源**

- 中山大学附属第三医院：邹小兵专家介绍，提到 BSR 模式和干预三原则。
- 邹小兵，《孤独症谱系障碍干预原则与 BSR 模式》，《中国儿童保健杂志》2019。
- 《自闭症儿童社交能力阶梯训练：BSR 模式》。

## Skill 2：郭延庆 ABA/核心行为信息视角

**核心用途**：把问题转成可观察行为，做 ABC 功能分析，再设计替代行为、辅助、强化和泛化。

**理论抓手**

- 应用行为分析关注行为前因、行为、后果和行为功能。
- 问题行为既可能是能力缺陷，也可能被后果维持。
- 家庭干预要把目标变成可练习、可记录、可强化的小行为。

**输出重点**

- 可能原因：逃避、获得、注意、自我刺激/自我调节或多重功能。
- 处理原则：目标行为、替代行为、辅助、强化、撤辅助、泛化。
- 干预动作：任务拆小、即时强化、教替代沟通、记录频率/时长/辅助等级。

**主要来源**

- 北京大学第六医院：郭延庆专家介绍，专业方向包括儿童孤独症和行为分析。
- 郭延庆，《应用行为分析与儿童行为管理》，华夏出版社。
- 郭延庆关于“正常化发展”和泛化目标的公开家长讲座整理。

## Skill 3：Steven Gutstein RDI 关系发展视角

**核心用途**：关注孩子在动态变化、共同调节、经验分享和弹性思维中的困难，让家长成为有意识的引导者。

**理论抓手**

- RDI 强调 Mindful Guiding Relationship。
- 家长在日常活动中通过渐进、可调的互动帮助孩子发展动态智能。
- 目标不是机械服从，而是在关系中处理不确定和变化。

**输出重点**

- 可能原因：动态智能不足、共同调节不足、经验分享困难、关系动机弱。
- 处理原则：少测试、多分享，慢节奏，可预测框架中加入微变化。
- 干预动作：共同活动、微小变化、共同解决问题、每天 7 天关系练习。

**主要来源**

- RDIconnect 官方网站和 RDI Program Mission。
- Steven E. Gutstein, “Empowering Families through Relationship Development Intervention,” Annals of Clinical Psychiatry, 2009。
- Steven E. Gutstein, The RDI Book, 2009。

## Skill 4：Michelle Garcia Winner 社交思维视角

**核心用途**：帮助孩子理解情境、他人想法、社交线索和自己行为的影响，把隐性社交规则显性化。

**理论抓手**

- Social Thinking Methodology 强调社会观察、观点采择、情境期待和社交问题解决。
- 干预常使用社交词汇、社交地图、思维泡泡、脚本和复盘。
- 适用于有社交学习差异的人群，不限于 ASD。

**输出重点**

- 可能原因：没读懂场景、他人想法、隐藏规则或行为影响。
- 处理原则：先教观察和理解，再练回应行为。
- 干预动作：社交地图、短脚本、情境演练、回家复盘。

**主要来源**

- Social Thinking 官方 “What is Social Thinking?”。
- Michelle Garcia Winner 与 Pamela Crooke，《You Are a Social Detective!》。
- Crooke & Winner 关于 Social Thinking 证据基础的文章。

## Skill 5：Stanley Greenspan DIR/Floortime 发展关系视角

**核心用途**：从发展阶段、个体差异和关系互动理解问题，通过跟随兴趣和情绪调节提升参与、沟通和思考。

**理论抓手**

- DIR = Developmental, Individual-differences, Relationship-based。
- Floortime 用尊重、游戏、愉悦和投入的互动促进发展。
- 目标包括自我调节、参与、双向沟通、共同解决问题、创造性思考。

**输出重点**

- 可能原因：调节困难、感官差异、沟通圈不足、关系参与低。
- 处理原则：先跟随，再挑战；先关系，再要求；先调节，再学习。
- 干预动作：地板时光片段、沟通圈扩展、感官准备、参与时长记录。

**主要来源**

- ICDL “What is Floortime?”。
- ICDL Clinical Practice Guidelines。
- ICDL 推荐书单：《Engaging Autism》《The Child with Special Needs》。

## 后续调用建议

推荐后端或前端按下面流程调用：

1. 用户输入：年龄、场景、具体行为、触发因素、持续时间、已尝试方法、安全风险。
2. 逐个 Skill 调用 `promptTemplate`，注入同一个 `case`。
3. 要求每个 Skill 输出同样结构：`possibleReasons`、`principles`、`interventions`、`sevenDayPlan`、`riskBoundary`。
4. 综合模块只做整合，不抹平差异：保留每个流派的独特逻辑。
5. 所有输出底部固定显示安全边界。

示例：

```js
import {
  autismInterventionSkills,
  autismSkillSafetyPolicy,
} from "./src/data/autismInterventionSkills.js";

const caseText = "孩子在商场听到广播和人声会捂耳朵尖叫，推开家长，不肯排队等待。";
const prompts = autismInterventionSkills.map((skill) =>
  skill.promptTemplate.replace("{{case}}", caseText)
);
```
